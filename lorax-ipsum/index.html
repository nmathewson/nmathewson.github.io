By the way, be careful when you tweet the "lorem ipsum" text or anything like it. I did so once, and immediately got followed by dozens of maddened spambots.

My working theory is:

  1. Somebody is selling a spambot that the user needs to configure with a list of search terms.
  2. The default list of search terms includes some or all of the lorem ipsum text.
  3. At least some users never actually change the configuration.

This was a few years back, so maybe those bots are gone now. Why not tweet some lorem ipsum and find out?
